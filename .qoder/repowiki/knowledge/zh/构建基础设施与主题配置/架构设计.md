- **构建流水线集成**：通过 `scripts/build-electron.mjs` 使用 esbuild 编译主进程与预加载脚本，并处理 Next.js standalone 模式的符号链接问题。
- **打包后处理钩子**：`after-pack.js` 针对 electron-builder 的局限，强制重建 better-sqlite3 以匹配目标 Electron ABI；`after-sign.js` 实现 macOS 的分层代码签名（从二进制到 Framework 再到主应用），支持开发者 ID 验证与 Ad-hoc 签名回退。
- **资产生成工具**：提供基于 sharp 的脚本（`generate-app-icon.mjs`, `gen-tray-icon.mjs`）自动化生成符合 macOS/Windows 规范的图标集及状态栏模板。
- **质量门禁**：包含针对 Git 钩子环境变量（`lint-hooks.mjs`）和文档索引一致性（`lint-docs-drift.mjs`）的静态检查逻辑。
- **主题数据层**：`themes/` 目录存储标准化的 JSON 格式主题配置，定义了 OKLCH 色彩空间下的 UI 语义变量及对应的 Shiki 代码高亮主题映射。
