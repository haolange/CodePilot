- 所有暴露给渲染进程的 API 必须通过 `preload.ts` 中的 `contextBridge.exposeInMainWorld` 进行封装，禁止在渲染层直接调用 `ipcRenderer`。
- 涉及文件系统路径或环境变量的操作需兼容多平台差异，例如 Windows 下优先探测 Git Bash，macOS/Linux 下通过登录 Shell 加载完整环境变量。
- 主进程产生的持久化日志在写入磁盘前必须经过 `sanitizeLogLine` 处理，以屏蔽 API 密钥、认证令牌及用户家目录路径等敏感信息。
- 子进程（如 Next.js 服务或安装向导）的生命周期管理需包含超时检测与强制清理逻辑，防止在主进程退出时产生僵尸进程。
